//// RESERVAS ////
- El cliente puede reservar cualquier producto en stock
- Para las reservas de productos disponibles, habrá un tiempo de caducidad, al cabo del cual el producto se libera y la seña se pierde.
- El cliente subirá al Sistema el comprobante de la seña
- El administrador del Sistema verificará el arribo de los fondos en las cuentas bancarias y confirmará la seña (a partir de este momento comienzan a corer los días de caducidad)
- Si el cliente se arrepiente, pierde la seña
- Al concretarse la venta, el administrador marcará la reserva como completada
- Posibilidad que el administrador extienda los días de la reserva ante el pedido del cliente

//// PEDIDOS ////
- El cliente puede pedir un producto que se encuentre en otra sucursal o proveedor
- Para pedir un producto el cliente debe señarlo con un porcentaje x del valor de venta
- El cliente debe cargar en el sistema el comprobante de la seña o notificar a algún administrador
- El administrador confirmará la recepción de la seña al revisar que el dinero llegue a las cuentas bancarias
- El administrador notificará al cliente cuando el producto solicitado haya sido pedido
- El administrador notificará al cliente y pedirá su confirmación si el producto no está disponible o no hay el color/variante requerida
- El administrador notificará al cliente cuando su producto arribe a la sucursal para que el cliente programe una cita. A partir de este momento el cliente tiene un tiempo limitado de días para cancelar la deuda y retirar el producto
